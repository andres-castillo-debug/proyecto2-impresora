PrintOS — Sistema de Cola de Impresión con Prioridad

Simulador de cola de impresión que implementa un Montículo Binario (Min-Heap)
como cola de prioridad y una Tabla de Dispersión (HashTable) para acceso O(1)
a usuarios, permitiendo gestionar documentos con distintos niveles de prioridad.
